## Cygwin

1. Install cygwin from [here](https://cygwin.com/install.html). 

2. Install following packages 

* bash-completion
* fzf-zsh
* fzf-zsh-completion
* curl
* tmux
* vim
* make
* perl
* tar
* tig
* unzip
* wget
* zip 
* zsh

3. Install fonts 

```bash
$ git clone https://github.com/powerline/fonts.git --depth=1
$ cd fonts 
$ ./install.sh
```
